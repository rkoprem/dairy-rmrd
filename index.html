One-time Setup
Google Sheets-ஐ database-ஆக connect பண்ணணும்
1
Google Sheets திற
sheets.new போய் புதிய Sheet create பண்ணுங்க.
Sheet name: Dairy RMRD என்று வையுங்க.
2
Apps Script திற
Sheet-ல் மேலே Extensions → Apps Script click பண்ணுங்க.
திறக்கும் editor-ல் உள்ள code-ஐ எல்லாம் delete பண்ணிவிட்டு, கீழே உள்ள "Copy Script" button தட்டி paste பண்ணுங்க.
பிறகு 💾 Save (Ctrl+S) பண்ணுங்க.
3
Deploy பண்ணு
Apps Script-ல் Deploy → New deployment click பண்ணுங்க.
• Type: Web app
• Execute as: Me
• Who has access: Anyone
Authorize பண்ணி Deploy பண்ணுங்க.
கிடைக்கும் Web app URL-ஐ copy பண்ணுங்க.
4
URL paste பண்ணு
Copy பண்ணிய URL-ஐ கீழே paste பண்ணி Connect பண்ணுங்க.
URL சரியாக இல்ல — script.google.com/macros/s/ என்று start ஆகணும்
